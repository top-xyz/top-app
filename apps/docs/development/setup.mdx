# Development Setup

This guide will help you set up your development environment to contribute to Top.

## Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v18 or later)
- pnpm (v8 or later)
- Git
- Docker (for running services locally)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/top-dev/top.git
cd top
```

2. Install dependencies:
```bash
pnpm install
```

3. Set up environment variables:
```bash
cp .env.example .env
```

Edit `.env` with your local configuration.

## Development Workflow

1. Start the development server:
```bash
pnpm dev
```

2. Run tests:
```bash
pnpm test
```

3. Build for production:
```bash
pnpm build
```

## Project Structure

```
top/
├── apps/           # Application packages
│   ├── web/        # Web application
│   ├── docs/       # Documentation site
│   └── api/        # API server
├── packages/       # Shared packages
│   ├── core/       # Core utilities
│   ├── ui/         # UI components
│   └── config/     # Shared configuration
└── scripts/        # Development scripts
```

## Development Tools

- **Code Editor**: We recommend using VS Code with our recommended extensions
- **API Testing**: Use Postman or Insomnia for API testing
- **Database**: PostgreSQL for primary data storage
- **Caching**: Redis for caching and real-time features
