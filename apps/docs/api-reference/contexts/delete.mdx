# Delete Context

Delete an existing context by its ID.

## Request

<ParamField path="contextId" type="string" required>
  The unique identifier of the context to delete
</ParamField>

### Example Request

```bash
curl -X DELETE https://api.top.dev/v1/contexts/ctx_123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

<ResponseField name="success" type="boolean">
  Whether the deletion was successful
</ResponseField>

<ResponseField name="id" type="string">
  The ID of the deleted context
</ResponseField>

### Example Response

```json
{
  "data": {
    "success": true,
    "id": "ctx_123"
  },
  "meta": {
    "requestId": "req_123",
    "timestamp": "2024-01-15T12:00:00Z"
  }
}
```

## Errors

<ResponseField name="401" type="object">
  Unauthorized - Invalid or missing API key
</ResponseField>

<ResponseField name="403" type="object">
  Forbidden - Insufficient permissions
</ResponseField>

<ResponseField name="404" type="object">
  Not Found - Context does not exist
</ResponseField>

<ResponseField name="409" type="object">
  Conflict - Context cannot be deleted in its current state
</ResponseField>

<ResponseField name="429" type="object">
  Too many requests - Rate limit exceeded
</ResponseField>

<ResponseField name="500" type="object">
  Internal server error
</ResponseField>
