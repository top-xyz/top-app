# Welcome to Top

Top is a revolutionary development environment that transforms how you interact with code. Through natural conversation and context-aware AI, Top helps you understand, create, and evolve code more efficiently than ever before.

## Key Features

### ü§ù Context-Aware Development

```typescript
// Top understands your code in context
const context = await top.createContext({
  intention: "Add authentication",
  codebase: "./src",
  requirements: ["OAuth", "Session management"]
})

// Natural conversation drives development
const result = await context.discuss(`
  I want to add user authentication using Auth0.
  It should handle both social and email login.
`)
```
